JAVA可视化闹钟源码

功能介绍：

1、可增加闹钟

2、可删除闹钟

3、时间到了响铃

4、关闭闹钟不会丢失闹钟（因为闹钟存储在txt文件中，不会因程序关闭就终止）

缺点

1、没有使用多线程，闹钟响起时只能等待1分钟或者关闭程序

2、界面设计不够美观，后期有时间会进行修改，重新设计

3、没有闹钟修改的功能，虽然可以通过增删来达到修改的目的，但功能仍然属于空缺范围
