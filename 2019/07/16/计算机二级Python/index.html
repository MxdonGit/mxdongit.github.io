и®Ўз®—жңәдәҢзә§еңЁиҝ‘дёӨе№ҙж–°еҠ дәҶpythonзҡ„йҖүжӢ©пјҢи¶ҒжңәиҖғдәҶдёҖдёӢпјҢйЎәдҫҝи®°еҪ•дёҖдёӢеӯҰд№ зҡ„дёҖдәӣжүҖиҺ·

зЁӢеәҸи®ҫи®ЎиҜӯиЁҖжҰӮиҝ°

иҖғзәІиҖғзӮ№пјҡ

иҝҷдёҖйғЁеҲҶдё»иҰҒжҳҜд»Ӣз»Қи®Ўз®—жңәиҜӯиЁҖзҡ„е…¬е…ұеёёиҜҶпјҢдёҖдәӣе°қиҜ•жҲ‘е°ұжҢүз…§иҮӘе·ұзҡ„зҗҶи§Јж–№ејҸжқҘи®°еҝҶпјҡ

зЁӢеәҸи®ҫи®ЎиҜӯиЁҖвҖ”вҖ”и®Ўз®—жңәдёҺдәәзұ»д№Ӣй—ҙзҡ„зҝ»иҜ‘е®ҳпјҢжүҖд»Ҙз§°дёәдәӨдә’дҪ“зі»пјҢзҝ»иҜ‘е®ҳз”Ёзҡ„иҜӯиЁҖз§°дёәи®Ўз®—жңәзЁӢеәҸпјҢе°ұеғҸдәәзұ»зҡ„иҜӯиЁҖжңүиҜӯжі•зӯүзӯүи§„еҲҷдёҖж ·гҖӮ

й«ҳзә§зј–зЁӢиҜӯиЁҖеҢ…жӢ¬йқҷжҖҒиҜӯиЁҖе’Ңи„ҡжң¬иҜӯиЁҖпјҢpythonжҳҜи§ЈйҮҠиҜӯиЁҖпјҢе°ұжҳҜзӣҙжҺҘи·ҹи®Ўз®—жңәеҜ№иҜқзҡ„йӮЈз§ҚпјҢд№ҹе°ұжҳҜи„ҡжң¬иҜӯиЁҖпјӣJavaйӮЈз§ҚжҳҜйңҖиҰҒзј–иҜ‘жүҚиғҪжү§иЎҢзҡ„иҜӯиЁҖпјҢе°ұзӣёеҪ“дәҺйңҖиҰҒдәҢж¬Ўзҝ»иҜ‘пјҢжҳҜйқҷжҖҒиҜӯиЁҖпјҢеӣ дёәйқҷжҖҒпјҲжҮ’пјүпјҢиҝҳйңҖиҰҒдёҖдёӘдј иҜқзҡ„жүҚиЎҢгҖӮ

е®ҳж–№з”ЁиҜӯзҡ„иҜқпјҢзј–иҜ‘жҳҜе°Ҷжәҗд»Јз ҒиҪ¬жҚўжҲҗзӣ®ж Үд»Јз Ғзҡ„иҝҮзЁӢпјҢи§ЈйҮҠжҳҜе°Ҷжәҗд»Јз ҒйҖҗжқЎиҪ¬жҚўжҲҗзӣ®ж Үд»Јз Ғзҡ„еҗҢж—¶йҖҗжқЎиҝҗиЎҢзӣ®ж Үд»Јз Ғзҡ„иҝҮзЁӢгҖӮдәҢиҖ…еҢәеҲ«еҸҜи§ҒдёӢеӣҫжқҘзҗҶи§Јпјҡ

зј–иҜ‘жҳҜдёҖж¬ЎжҖ§зҝ»иҜ‘пјҢеҶҚдј иҜқпјҢз»ҸиҝҮдј иҜқдәәд»ҘеҗҺпјҢе°ұдёҚйңҖиҰҒзҝ»иҜ‘е®ҳеҶҚзҝ»иҜ‘дёҖйҒҚдәҶпјҢжүҖд»ҘдёҖж—ҰзЁӢеәҸиў«зј–иҜ‘пјҢе°ұдёҚйңҖиҰҒеҶҚж¬Ўзј–иҜ‘пјҢжүҖд»ҘйҮҚеӨҚжү§иЎҢйҖҹеәҰжӣҙеҝ«пјҢеҪ“з„¶иҝҷдёӘзӣ®ж Үд»Јз Ғд№ҹе°ұдёҚеҶҚйңҖиҰҒзј–иҜ‘еҷЁгҖӮ

и§ЈйҮҠжҳҜжҜҸж¬ЎйғҪйңҖиҰҒи§ЈйҮҠпјҢеӣ дёәзҝ»иҜ‘дёҖйҒҚзҡ„еӨ–иҜӯе®№жҳ“еҝҳпјҢе‘ҠиҜүдј иҜқдәәд»ҘеҗҺпјҢдј иҜқдәәе°ұдёҚе®№жҳ“еҝҳиҮӘе·ұжҜҚиҜӯи®°еҝҶзҡ„еҶ…е®№пјҢдҪҶжҳҜи§ЈйҮҠжҳҜзҝ»иҜ‘е®ҳе…јиҒҢдј иҜқдәәпјҢжүҖд»ҘеҸӘиҰҒзҝ»иҜ‘е®ҳеӯҳеңЁпјҢзЁӢеәҸе°ұеҸҜд»Ҙжү§иЎҢгҖӮж— и®әжҳҜйӮЈз§Қзі»з»ҹпјҢиҖҢдё”дёҖдёӘдәәе№ІпјҢзЁӢеәҸзә й”ҷе’Ңз»ҙжҠӨд№ҹйқһеёёж–№дҫҝгҖӮ

PythonиҜӯиЁҖжҰӮиҝ°

иҝҷжҳҜpythonзҡ„жңҖе°ҸзЁӢеәҸдәҶгҖӮ

иҮідәҺpythonзҡ„ејҖеҸ‘зҺҜеўғй…ҚзҪ®иҝҷйҮҢе°ұдёҚд»Ӣз»ҚдәҶпјҢзҺҜеўғеҸҳйҮҸеҸҜд»ҘзӣҙжҺҘж·»еҠ пјҢжҜ”jdkж–№дҫҝеӨҡдәҶ

pythonзҡ„и§ЈйҮҠеҷЁжңүдёӨдёӘйҮҚиҰҒе·Ҙе…·пјҡIDLEвҖ”вҖ”pythonйӣҶжҲҗејҖеҸ‘зҺҜеўғпјҲзӣёеҪ“дәҺдәәзұ»гҖҒзҝ»иҜ‘е®ҳгҖҒи®Ўз®—жңәз»„жҲҗзҡ„ж•ҙдҪ“пјүпјҢpipвҖ”вҖ”pythonз¬¬дёүж–№еә“е®үиЈ…е·Ҙе…·пјҲйӮҖиҜ·зҝ»иҜ‘е®ҳзҡ„еҠҹиғҪпјү

PythonзЁӢеәҸзҡ„иҝҗиЎҢж–№ејҸвҖ”вҖ”дәӨдә’ејҸпјҲе‘Ҫд»ӨиЎҢпјүгҖҒж–Үд»¶ејҸпјҲйқһе‘Ҫд»ӨиЎҢпјү

зЁӢеәҸзҡ„еҹәжң¬зј–еҶҷж–№жі•

IPOзЁӢеәҸзј–еҶҷж–№жі•вҖ”вҖ”InputпјҲиҫ“е…ҘпјҢзӣёеҪ“дәҺдәәзұ»зј–еҶҷд»Јз ҒиҝҷдёӘиҝҮзЁӢпјүпјҢprocessпјҲеӨ„зҗҶпјҢзӣёеҪ“дәҺзҝ»иҜ‘пјүпјҢoutputпјҲи®Ўз®—жңәеҫ—еҲ°зҝ»иҜ‘е®ҳз»ҷзҡ„дҝЎжҒҜеҒҡеҮәе“Қеә”пјү

PythonзЁӢеәҸзҡ„зү№зӮ№

Pythonе…·жңүйҖҡз”ЁжҖ§пјҢеӣ дёәеә”з”ЁйўҶеҹҹе№ҝпјҢжүҖд»ҘжүҚиҰҒеӯҰд№ пјҢеҪ“дҪңе·Ҙе…·з”Ёд№ҹеҘҪ

PythonиҜӯжі•з®ҖжҙҒпјҢеҸӘжңү33дёӘдҝқз•ҷеӯ—пјҢзңӢиө·жқҘе°ұеғҸиӢұиҜӯжӯЈеёёиЎЁиҫҫдёҖж ·

Pythonз”ҹжҖҒй«ҳдә§пјҢеә“зү№еҲ«еӨҡпјҢд№ҹе°ұжҳҜзҝ»иҜ‘е®ҳзү№еҲ«е……и¶іпјҢеҫҲеӨҡиҚ’иҜһж— зҗҶзҡ„иҰҒжұӮйғҪеҸҜд»Ҙиў«зҝ»иҜ‘е®ҳеҗ¬жҮӮ

йҷӨдәҶPythonиҜӯжі•зҡ„дёүдёӘйҮҚиҰҒзү№зӮ№еӨ–пјҢPythonзЁӢеәҸиҝҳ жңүдёҖдәӣе…·дҪ“зү№зӮ№гҖӮ

вҖўе№іеҸ°ж— е…іпјҡеүҚйқўиҜҙеҲ°иҝҮпјҢеҸӘиҰҒжңүзҝ»иҜ‘е®ҳе°ұеҸҜд»ҘпјҢеңЁд»»дҪ•зі»з»ҹйғҪеҸҜд»ҘиҝҗиЎҢ

вҖўејәеҲ¶еҸҜиҜ» пјҡд№ҹе°ұжҳҜеҲ©з”Ёзј©иҝӣжқҘиЎЁжҳҺйҖ»иҫ‘е…ізі»пјҢзӣёеҪ“дәҺеҸҘеӯҗд№Ӣй—ҙжІЎжңүеҒңйЎҝпјҢдҪҝз”Ёзј©иҝӣжқҘејәеҲ¶зҗҶи§Ј

вҖўж”ҜжҢҒдёӯж–ҮпјҡзңӢдёӢеӣҫ

PythonиҜӯиЁҖеҹәжң¬иҜӯжі•е…ғзҙ 

иҖғзәІиҖғзӮ№пјҡ

зЁӢеәҸзҡ„ж јејҸжЎҶжһ¶

зј©иҝӣпјҡ

PythonиҜӯиЁҖйҮҮз”ЁдёҘж јзҡ„вҖңзј©иҝӣвҖқжқҘиЎЁжҳҺзЁӢеәҸзҡ„ ж јејҸжЎҶжһ¶пјҢз”ЁжқҘиЎЁзӨәд»Јз Ғд№Ӣй—ҙзҡ„еҢ…еҗ«е’ҢеұӮж¬Ўе…ізі»гҖӮ 1дёӘзј©иҝӣ = 4дёӘз©әж ј =1дёӘйҖҖж јгҖӮдҪҶжҳҜз©әж је’ҢйҖҖж јдёҚиғҪж··з”ЁгҖӮиҖҢдё”зј©иҝӣжҳҜPythonиҜӯиЁҖдёӯиЎЁжҳҺзЁӢеәҸжЎҶжһ¶зҡ„е”ҜдёҖжүӢж®өгҖӮе…¶йҮҚиҰҒжҖ§еҸҜи§ҒдёҖж–‘гҖӮ

жіЁйҮҠпјҡ

жіЁйҮҠе°ұжҳҜеңЁиҫ“е…Ҙзҡ„ж—¶еҖҷеҠ е…ҘдёҖдәӣдёҚжғіи®©зҝ»иҜ‘е®ҳзҝ»иҜ‘зҡ„дёңиҘҝпјҢдҪҶжҳҜзҝ»иҜ‘е®ҳеҰӮдҪ•еҺ»еҢәеҲҶе“ӘдәӣжҳҜиҰҒзҡ„пјҢе“ӘдәӣжҳҜдёҚиҰҒзҡ„е‘ўпјҹжүҖд»Ҙе°ұжңүдәҶжіЁйҮҠпјҢеҚ•иЎҢжіЁйҮҠз”Ёдә•еҸ·гҖӮеӨҡиЎҢжіЁйҮҠз”ЁдёүеҜ№еј•еҸ·пјҢдёүйҳҹеј•еҸ·д№Ӣй—ҙзҡ„еҶ…е®№е°ұжҳҜиў«жіЁйҮҠжҺүдәҶгҖӮеҚ•еј•еҸ·еҸҢеј•еҸ·ж— жүҖи°“гҖӮ

еҸҳйҮҸпјҡ

еҸҳйҮҸе°ұжҳҜдёҖдёӘдёҖдёӘзҡ„е°ҸжҲҝеӯҗпјҢз”ЁжқҘеӯҳж”ҫдёңиҘҝзҡ„пјҢз»Ҹеёёз”ЁдёҖдёӘзӯүеҸ·жҠҠдёңиҘҝж”ҫиҝӣеҸҳйҮҸпјҲжҲҝеӯҗпјүйҮҢпјҢд№ҹе°ұжҳҜиөӢеҖјгҖӮ

е‘ҪеҗҚпјҡ

PythonиҜӯиЁҖе…Ғи®ёйҮҮз”ЁеӨ§еҶҷеӯ—жҜҚгҖҒе°ҸеҶҷеӯ—жҜҚгҖҒж•° еӯ—гҖҒдёӢеҲ’зәҝ(_)е’Ңжұүеӯ—зӯүеӯ—з¬ҰеҸҠе…¶з»„еҗҲз»ҷеҸҳйҮҸе‘Ҫ еҗҚпјҢдҪҶеҗҚеӯ—зҡ„йҰ–еӯ—з¬ҰдёҚиғҪжҳҜж•°еӯ—пјҢдёӯй—ҙдёҚиғҪеҮәзҺ°з©әж јпјҢй•ҝеәҰжІЎжңүйҷҗеҲ¶ nгҖӮжіЁж„Ҹпјҡж ҮиҜҶз¬ҰеҜ№еӨ§е°ҸеҶҷж•Ҹж„ҹпјҢpythonе’ҢPython жҳҜдёӨдёӘдёҚеҗҢзҡ„еҗҚеӯ—

дҝқз•ҷеӯ—пјҡ

pythonдёӯдҪ зӣ–зҡ„жҲҝеӯҗжҳҜеҸҳйҮҸпјҢж—©е°ұеӯҳеңЁзҡ„еҗҚиғңе»әзӯ‘е°ұжҳҜеҸ«еҒҡдҝқз•ҷеӯ—дәҶпјҢиҮӘе·ұзҡ„жҲҝеӯҗзҲұе’Ӣе’Ӣең°пјҢеҗҚиғңеҸӨиҝ№еҸҜдёҚиғҪд№ұеҠЁпјҢиҖҢдё”иҝҳдёҚиғҪе»әдёҖдёӘе’ҢеҗҚиғңе»әзӯ‘дёҖж ·зҡ„жҲҝеӯҗпјҢиҰҒдёҚз„¶е°ұдҫөжқғдәҶпјҢиҰҒеқҗзүўпјҲеҮәbugпјүзҡ„пјҒ

ж•°жҚ®зұ»еһӢ

PythonиҜӯиЁҖж”ҜжҢҒеӨҡз§Қж•°жҚ®зұ»еһӢпјҢжңҖз®ҖеҚ•зҡ„еҢ…жӢ¬ж•°еӯ—зұ»еһӢгҖҒеӯ—з¬ҰдёІзұ»еһӢпјҢз•Ҙеҫ®еӨҚжқӮзҡ„еҢ…жӢ¬е…ғз»„зұ»еһӢгҖҒйӣҶеҗҲзұ»еһӢгҖҒеҲ—иЎЁзұ»еһӢгҖҒеӯ—е…ёзұ»еһӢзӯүгҖӮ

ж•°еӯ—зұ»еһӢ

иЎЁзӨәж•°еӯ—жҲ–ж•°еҖјзҡ„ж•°жҚ®зұ»еһӢз§°дёәж•°еӯ—зұ»еһӢпјҢPythonиҜӯиЁҖжҸҗдҫӣ3з§Қж•°еӯ—зұ»еһӢпјҡж•ҙж•°гҖҒжө®зӮ№ж•°е’ҢеӨҚж•°пјҢеҲҶеҲ«еҜ№еә”ж•°еӯҰдёӯзҡ„ж•ҙж•°гҖҒе®һж•°е’ҢеӨҚж•°гҖӮ

дёҖдёӘж•ҙж•°еҖјеҸҲеҸҜд»ҘиЎЁзӨәдёәеҚҒиҝӣеҲ¶гҖҒеҚҒе…ӯиҝӣеҲ¶гҖҒе…«иҝӣеҲ¶е’ҢдәҢиҝӣеҲ¶зӯүдёҚеҗҢиҝӣеҲ¶еҪўејҸгҖӮдёҖдёӘжө®зӮ№ж•°еҸҜд»ҘиЎЁзӨәдёәеёҰжңүе°Ҹж•°зӮ№зҡ„дёҖиҲ¬еҪўејҸпјҢд№ҹеҸҜд»ҘйҮҮз”Ёз§‘еӯҰи®Ўж•°жі•иЎЁзӨәгҖӮ

еӯ—з¬ҰдёІ

иҮідәҺеӯ—з¬ҰдёІпјҢ PythonиҜӯиЁҖдёӯпјҢеӯ—з¬ҰдёІжҳҜз”ЁдёӨдёӘеҸҢеј•еҸ·вҖң вҖқжҲ–иҖ…еҚ• еј•еҸ·вҖҳ вҖҷжӢ¬иө·жқҘзҡ„дёҖдёӘжҲ–еӨҡдёӘеӯ—з¬ҰгҖӮ

еҸҜд»ҘйҮҮз”Ё[N: M]пјҲе·Ұй—ӯеҸіејҖеҢәй—ҙпјүж јејҸиҺ·еҸ–еӯ—з¬ҰдёІзҡ„еӯҗдёІпјҢиҝҷдёӘж“ҚдҪңиў«еҪўиұЎең°з§°дёәеҲҮзүҮгҖӮ

еҸҜд»ҘйҖҡиҝҮPythonй»ҳи®ӨжҸҗдҫӣзҡ„len()еҮҪж•°иҺ·еҸ–еӯ—з¬ҰдёІ зҡ„й•ҝеәҰпјҢдёҖдёӘдёӯж–Үеӯ—з¬Ұе’ҢиҘҝж–Үеӯ—з¬Ұзҡ„й•ҝеәҰйғҪи®°дёә1гҖӮ

зЁӢеәҸзҡ„иҜӯеҸҘе…ғзҙ 

иЎЁиҫҫејҸ

дә§з”ҹжҲ–и®Ўз®—ж–°ж•°жҚ®еҖјзҡ„д»Јз ҒзүҮж®өз§°дёәиЎЁиҫҫејҸпјҢз®ҖеҚ•иҜҙе°ұжҳҜиҝһжҺҘеңЁдёҖиө·зҡ„дёҖдёӘеҸҘеӯҗ/зҹӯеҸҘ

иөӢеҖј
